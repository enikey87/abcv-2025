# ABC/VEN-анализ лекарственных средств

Система для фармакоэкономического анализа расходов медицинского учреждения на лекарственные препараты.

## Что такое ABC/VEN-анализ?

**ABC/VEN-анализ** — это комбинированный метод оценки рациональности использования финансовых ресурсов на закупку лекарственных средств. Метод позволяет выявить приоритетные направления для оптимизации затрат и улучшить качество лекарственного обеспечения.

---

## ABC-анализ

ABC-анализ классифицирует лекарственные препараты **по доле в общих затратах** учреждения. Основан на принципе Парето (80/20).

### Категории ABC

| Группа | Доля в затратах | Доля в номенклатуре | Характеристика |
|--------|-----------------|---------------------|----------------|
| **A** | ~80% | 10–20% | Наиболее затратные препараты, требуют пристального контроля |
| **B** | ~15% | 20–30% | Препараты средней стоимости, стандартный контроль |
| **C** | ~5% | 50–70% | Недорогие препараты, упрощённый контроль |

### Методика расчёта

1. Рассчитать затраты на каждый препарат за период (цена × количество)
2. Отсортировать препараты по убыванию затрат
3. Рассчитать накопленный процент от общих затрат
4. Присвоить категории:
   - **A** — накопленный % до 80%
   - **B** — накопленный % от 80% до 95%
   - **C** — накопленный % от 95% до 100%

## VEN-анализ

VEN-анализ классифицирует препараты **по степени клинической значимости** (жизненной важности).

### Категории VEN

| Категория | Название | Описание | Примеры |
|-----------|----------|----------|---------|
| **V** | Vital (Жизненно важные) | Препараты для лечения угрожающих жизни состояний, без которых невозможно оказание помощи | Инсулин, адреналин, антибиотики для сепсиса, наркотические анальгетики, противошоковые средства |
| **E** | Essential (Необходимые) | Препараты для лечения менее тяжёлых, но значимых заболеваний | Антигипертензивные, НПВС, антигистаминные, большинство антибиотиков |
| **N** | Non-essential (Второстепенные) | Препараты с недоказанной эффективностью или для лечения лёгких состояний | Витамины, БАДы, гепатопротекторы, иммуномодуляторы |

### Критерии отнесения к категориям

Присвоение категории VEN основывается на:
- Включении в перечень ЖНВЛП (жизненно необходимых и важнейших лекарственных препаратов)
- Клинических рекомендациях и стандартах лечения
- Доказательной базе эффективности препарата
- Экспертной оценке врачебной комиссии учреждения

## Матрица ABC/VEN

Комбинация двух методов создаёт матрицу из 9 категорий, позволяющую принимать управленческие решения.

|       | V (Жизненно важные) | E (Необходимые) | N (Второстепенные) |
|-------|---------------------|-----------------|---------------------|
| **A** | AV | AE | AN ⚠️ |
| **B** | BV | BE | BN ⚠️ |
| **C** | CV | CE | CN |

### Интерпретация результатов

#### Приоритетные категории (обязательное обеспечение)
- **AV, BV, CV** — жизненно важные препараты всех ценовых категорий. Необходимо гарантированное наличие, недопустим дефицит.

#### Категории стандартного контроля
- **AE, BE, CE** — необходимые препараты. Стандартные процедуры закупки и контроля запасов.

#### Проблемные категории (требуют анализа)
- **AN** — высокие затраты на второстепенные препараты. Требуется пересмотр целесообразности закупок.
- **BN** — средние затраты на второстепенные препараты. Рекомендуется оптимизация.
- **CN** — низкие затраты на второстепенные препараты. Возможно исключение из формуляра.

## Цели и задачи анализа

### Основные цели

1. **Оптимизация бюджета** — выявление нерациональных расходов и перераспределение средств в пользу жизненно важных препаратов
2. **Повышение качества лекарственной помощи** — гарантированное обеспечение препаратами категорий V и E
3. **Формирование формуляра** — обоснованный выбор препаратов для включения в формулярный перечень учреждения

### Практические задачи

- Выявление препаратов с завышенными затратами (группа A)
- Обнаружение нерациональных закупок (категории AN, BN)
- Контроль наличия жизненно важных препаратов (категория V)
- Обоснование заявок на финансирование
- Сравнительный анализ расходов по периодам
- Бенчмаркинг с другими учреждениями

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Команды запуска

```bash
# Анализ полного набора данных (включая Синагис)
npm run analyze -- assets/ОПН.csv

# Анализ без Синагиса (для типичного распределения ABC)
npm run analyze -- assets/ОПН_без_синагиса.csv

# Анализ произвольного файла
npm run analyze -- путь/к/файлу.csv
```

### Выходные файлы

Результаты сохраняются в папку `output/`:
- `{имя_файла}_abc_ven.csv` — таблица с ABC/VEN категориями
- `{имя_файла}_report.txt` — текстовый отчёт

## Исходные данные

Исходные данные для анализа находятся в файле `assets/ОПН.csv` (отчёт о продажах/расходах).

### Структура таблицы

| Колонка | Описание |
|---------|----------|
| Код | Внутренний код препарата |
| Товар | Наименование лекарственного средства |
| Ед. | Единица измерения (уп., фл., шт.) |
| Кол-во | Количество израсходованных единиц |
| Сумма | Стоимость в рублях |
| VEN | Категория жизненной важности |

### VEN-классификация в данных

| Значение | Категория | Описание |
|----------|-----------|----------|
| **V** | Vital | Жизненно важные препараты |
| **E** | Essential | Необходимые препараты |
| **N** | Non-essential | Второстепенные препараты |

## Источники данных

- Данные бухгалтерского учёта о закупках ЛС
- Аптечный учёт движения препаратов
- Перечень ЖНВЛП
- Формулярный перечень учреждения
- Клинические рекомендации Минздрава РФ

---

## Нормативная база

- Федеральный закон № 61-ФЗ «Об обращении лекарственных средств»
- Распоряжение Правительства РФ об утверждении перечня ЖНВЛП
- Приказы Минздрава РФ о порядке формирования формулярных перечней
