# Формат вывода ABC/VEN анализа

Формат основан на таблицах из статьи:
https://clinpharm-journal.ru/articles/2024-2/abc-ven-analiz-potrebleniya-lekarstvennyh-preparatov-v-mnogoprofilnom-statsionare/

---

## Таблица 1: ABC-анализ

Показывает распределение препаратов по ABC-группам.

| Группа | Число МНН | % МНН | Затраты, руб. | % затрат |
|--------|-----------|-------|---------------|----------|
| A      | N         | X%    | XXX XXX.XX    | ~80%     |
| B      | N         | X%    | XXX XXX.XX    | ~15%     |
| C      | N         | X%    | XXX XXX.XX    | ~5%      |
| Итого  | N         | 100%  | XXX XXX.XX    | 100%     |

---

## Таблица 2: VEN-анализ

Показывает распределение препаратов по VEN-категориям.

| Категория | Число МНН | % МНН | Затраты, руб. | % затрат |
|-----------|-----------|-------|---------------|----------|
| V         | N         | X%    | XXX XXX.XX    | X%       |
| E         | N         | X%    | XXX XXX.XX    | X%       |
| N         | N         | X%    | XXX XXX.XX    | X%       |
| Итого     | N         | 100%  | XXX XXX.XX    | 100%     |

---

## Таблица 3: Матрица ABC/VEN (% затрат внутри группы)

Показывает распределение затрат на V/E/N препараты **внутри каждой ABC-группы**.
Сумма процентов по строке = 100%.

| Группа | V, %   | E, %   | N, %   |
|--------|--------|--------|--------|
| A      | XX.XX% | XX.XX% | XX.XX% |
| B      | XX.XX% | XX.XX% | XX.XX% |
| C      | XX.XX% | XX.XX% | XX.XX% |

**Важно**: проценты рассчитываются от суммы затрат **внутри ABC-группы**, а не от общей суммы.

---

## Пример вывода в консоль

```
═══════════════════════════════════════════════════════════════
                    ABC/VEN АНАЛИЗ
═══════════════════════════════════════════════════════════════

Всего позиций: 150
Общая сумма: 10 000 000.00 руб.

Таблица 1. ABC-анализ
┌────────┬───────────┬────────┬─────────────────┬──────────┐
│ Группа │ Число МНН │ % МНН  │ Затраты, руб.   │ % затрат │
├────────┼───────────┼────────┼─────────────────┼──────────┤
│ A      │        15 │  10.0% │   8 000 000.00  │   80.00% │
│ B      │        35 │  23.3% │   1 500 000.00  │   15.00% │
│ C      │       100 │  66.7% │     500 000.00  │    5.00% │
├────────┼───────────┼────────┼─────────────────┼──────────┤
│ Итого  │       150 │ 100.0% │  10 000 000.00  │  100.00% │
└────────┴───────────┴────────┴─────────────────┴──────────┘

Таблица 2. VEN-анализ
┌───────────┬───────────┬────────┬─────────────────┬──────────┐
│ Категория │ Число МНН │ % МНН  │ Затраты, руб.   │ % затрат │
├───────────┼───────────┼────────┼─────────────────┼──────────┤
│ V         │        30 │  20.0% │   3 000 000.00  │   30.00% │
│ E         │        80 │  53.3% │   5 500 000.00  │   55.00% │
│ N         │        40 │  26.7% │   1 500 000.00  │   15.00% │
├───────────┼───────────┼────────┼─────────────────┼──────────┤
│ Итого     │       150 │ 100.0% │  10 000 000.00  │  100.00% │
└───────────┴───────────┴────────┴─────────────────┴──────────┘

Таблица 3. Матрица ABC/VEN (% затрат внутри группы)
┌────────┬────────┬────────┬────────┐
│ Группа │ V, %   │ E, %   │ N, %   │
├────────┼────────┼────────┼────────┤
│ A      │  25.00 │  60.00 │  15.00 │
│ B      │  30.00 │  50.00 │  20.00 │
│ C      │  40.00 │  45.00 │  15.00 │
└────────┴────────┴────────┴────────┘

═══════════════════════════════════════════════════════════════
```
